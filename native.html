Data attributes:

Idea: define the same on scroll behaviour using smoothScrollbar on large devices and document scroll on small ones

  • data-os:attach watcher;
  • data-os-offset:override default offset (pixels);
  • data-os-action:define a function to trigger (class method, extendeded class method);
  • data-os-once:watch element once;
  • data-os-v-parallax:add vertical parallax effect;
  • data-os-h-parallax:add horizontal parallax effect;
